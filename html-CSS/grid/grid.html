Whilst flexbox is more suited to individual columns and rows - Grid is used for multiple columns and rows (Think instagram user profile)

Grid eliminates a lot of cruft code - less adding containers and class' means less that needs to be styled.

When setting grid template rows - Implicit and Explicit grid can be applied.

If I have a Grid container with 4 children but only specify 3 columns and not declared any rows the grid will create new rows as needed (IMPLICIT)

IMPLICIT means grid makes space automatically

EXPLICIT means declaring both colums and rows
